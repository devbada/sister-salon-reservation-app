## 버전 형식

- **Major.Minor.Patch** (예: 1.0.0)
- Major: 큰 기능 변경 또는 호환성 없는 변경
- Minor: 새 기능 추가 (하위 호환)
- Patch: 버그 수정 (하위 호환)
