주요 기능

살롱 운영에 필요한 모든 기능을 하나의 앱에서

📅

예약 관리

간편한 예약 생성, 수정, 삭제. 대기 → 확정 → 완료까지 5단계 상태 관리로 예약 현황을 한눈에.

👥

고객 관리

고객 정보와 방문 이력을 체계적으로 관리. 검색 및 필터링으로 빠르게 고객을 찾으세요.

✂️

디자이너 관리

스타일리스트별 전문 분야를 관리하고 예약 배정. 디자이너별 실적도 한눈에.

🕐

영업시간 관리

요일별 영업시간과 휴식시간 설정. 특별 휴무일 및 공휴일 관리도 가능합니다.

📊

통계 대시보드

일별, 월별 예약 통계와 차트로 비즈니스 인사이트를 한눈에 파악하세요.

🔒

앱 잠금 & 백업

PIN으로 앱을 보호하고, iCloud/Google Drive로 안전하게 백업하세요.

💻

멀티 플랫폼

iPhone, iPad, Mac, Windows, Android 어디서든 최적화된 반응형 UI.

📱

오프라인 우선

서버 없이 내 기기에서만 동작. 인터넷 연결 없이도 언제든 사용 가능.
